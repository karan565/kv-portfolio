## üöÄ Quick Start

### Viewing the Portfolio
Simply visit `https://karanvaghela.pages.dev/`
Or open `index.html` in any modern web browser.

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/karan565/kv-portfolio.git
   cd kv-portfolio
   ```

2. Open with a local server (optional but recommended):
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

### Deployment

The portfolio is a single HTML file and can be deployed to:
- **GitHub Pages**: Push to `gh-pages` branch
- **Netlify**: Drag and drop the file
- **Vercel**: Deploy directly from GitHub
- **Any static hosting**: Upload `index.html`

## üìù Customization

To customize this portfolio for your own use:

1. **Personal Information**: Update name, title, and subtitle in the header
2. **Contact Links**: Modify email, GitHub, LinkedIn, and phone number
3. **Resume Link**: Replace Google Drive link with your resume URL
4. **Experience Section**: Add/edit your work experience
5. **Projects**: Update with your personal projects
6. **Education**: Add your academic background
7. **Skills**: List your technical skills and tools
8. **Colors**: Modify CSS variables in `:root` for custom theme
